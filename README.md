<h1 align="center">Giwifi多功能自动处理程序</h1><br />
前言：Giwifi自动登录的程序编写的目的是解决登录繁琐、经常掉线的问题。通过一些技巧让登录变得更加简单。程序使用本地BW的浏览器，可能会导致内存不释放的问题，但是可以很好节约程序的大小。<br />
本页面将会继续完善。<br />

程序基于 C# 编写.
